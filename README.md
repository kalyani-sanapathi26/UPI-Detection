# Secure UPI Machine Learning-Driven Fraud Detection System

This project focuses on detecting fraudulent UPI transactions using advanced Machine Learning algorithms and ensemble learning techniques. The system analyzes transaction patterns and predicts whether a transaction is legitimate or fraudulent.


## Technologies Used
- Python
- Google Colab
- Pandas
- NumPy
- Scikit-Learn
- XGBoost
- LightGBM
- Matplotlib

## Dataset
Dataset collected from Kaggle Financial Transaction Dataset.


## Machine Learning Algorithms Implemented
- XGBoost
- Gradient Boosting
- AdaBoost
- Hybrid Ensemble Model
- LightGBM
- Extra Trees Classifier

## Explainable AI
Feature importance analysis was used to identify important transaction features contributing to fraud detection.

## Model Accuracy
| Algorithm | Accuracy |
|------------|------------|
| XGBoost | 99.12% |
| Gradient Boosting | 98.95% |
| AdaBoost | 97.84% |
| Hybrid Ensemble | 99.20% |
| LightGBM | 99.05% |
| Extra Trees | 98.77% |
